Fluxo de Trabalho
1

Usuario fornece PRD

Documento de requisitos do produto (PRD) e passado como argumento do comando de inicio.

2

/dev-team-start

Inicializa o projeto em 5 etapas sequenciais: cria estrutura board/ e docs/, PO analisa PRD e cria tarefas atomicas uma api/tela por vez, Tech Lead define stack com versoes exatas, DevOps faz scaffold do projeto, PO injeta contexto tecnico em cada tarefa.

PO Agent Tech Lead DevOps
3

/dev-team-run

Autonomo: /dev-team-run lanca tarefas independentes em paralelo usando worktrees isolados. Loop continuo ate tudo VERIFIED.

Backend Frontend DevOps Tech Lead
4

Implementacao + Testes

Cada agente dev le sua tarefa (board/todo/{ID}.md), move para in_progress/, implementa o codigo, escreve testes unitarios e de integracao, e move para done/ ao concluir. Decisoes tecnicas novas sao registradas em DECISIONS.md.

Backend Frontend
5

QA com Playwright E2E

QA Agent valida cada tarefa em done/: roda testes unitarios e de integracao, escreve specs E2E com Playwright, captura screenshots por criterio de aceite. Screenshots sao a fonte de verdade.

QA Agent
✓

Projeto Concluido

Todas as tarefas em board/verified/. Se bugs forem encontrados, QA cria tarefas FIX-* que retornam ao ciclo. O loop do /dev-team-run repete ate zero tarefas pendentes.

Ciclo de Vida da Tarefa
todo/
→
in_progress/
→
done/
→
verified/
blocked/  ← tarefa bloqueada  |  Tech Lead decide → volta para todo/  |  QA encontra bug critico → done/ volta para todo/ como FIX-*
Agentes

PO Agent

Product Owner
Analisa PRDs, cria tarefas atomicas com criterios de aceite testaveis, mapeia dependencias, injeta contexto tecnico.
Tools: Read Write Edit Glob

Tech Lead

Arquiteto
Define stack com versoes exatas, resolve conflitos tecnicos, desbloqueia tarefas BLOCKED, registra decisoes em DECISIONS.md.
Tools: Read Write Edit Glob Grep

Backend Agent

Desenvolvedor Backend
Implementa tarefas [BACK]: APIs, logica de negocio, banco de dados, integracoes. Escreve testes unitarios e de integracao.
Tools: Read Write Edit Bash Glob Grep

Frontend Agent

Desenvolvedor Frontend
Implementa tarefas [FRONT]: componentes, telas, integracao com APIs. Sempre trata 3 estados: loading, error, empty.
Tools: Read Write Edit Bash Glob Grep

QA Agent

QA Engineer
Valida tarefas DONE com 3 camadas: unit, integration, E2E Playwright. Screenshots sao prova de aceite. Cria tarefas FIX-* para bugs.
Tools: Read Write Edit Bash Glob Grep

DevOps Agent

Infraestrutura
Scaffold do projeto: configs, dependencias, scripts, Playwright setup, Docker. Garante que o projeto roda sem erros.
Tools: Read Write Edit Bash Glob Grep
Skills

task-reader

Le e parseia arquivo de tarefa board/{status}/{ID}.md. Extrai frontmatter, descricao, criterios, contexto e handoff.

task-writer

Atualiza tarefa e move entre pastas de status via git mv. Preenche Log, Handoff e PROGRESS.md.

board-scanner

Escaneia frontmatter de todos os arquivos do board. Resolve dependencias e ordena tarefas prontas.

QA Agent — fonte de verdade para aceite
Integration Tests

Unit Tests
Dev Agent — logica de negocio
